# Web Pop Skill

You are generating web pages for the **Power.Talks** project to display ERCOT research reports.

## Pages to Generate

### 1. Dashboard — `html/index.html`
A report index page with:
- Page title: "Power.Talks — ERCOT Reports"
- Filter bar: dropdown for category (All / MKT.RULES / STKHDR.MEETS), date range picker, keyword search box
- Report table: columns = Category, Document Name, Date, Summary snippet
- Each row links to its report viewer page (`report_<filename>.html`)
- Report data loaded from `html/reports-manifest.json` (generated by study.py)

### 2. Report Viewer — `html/report.html`
A two-panel layout for each report:
- **Left panel (60%):** Full report content rendered as formatted HTML
  - Show all 5 sections (Title, Executive Summary, Key Points, Market Impact, Action Items)
- **Right panel (40%):** Embedded Claude chat
  - Input box at the bottom: "Ask a question about this report..."
  - Chat history displayed above
  - On submit: call Anthropic API (`claude-sonnet-4-6`) with the report content as system context
  - Stream the response back into the chat panel
  - API key read from `window.ANTHROPIC_API_KEY` (set in `html/config.js`)

## Shared Files

### `html/style.css`
- Clean, professional look: white background, dark navy header (`#0a2540`), accent color `#2563eb`
- Responsive layout (works on desktop and tablet)
- Table rows highlight on hover
- Chat panel has a distinct light-grey background

### `html/app.js`
- `loadManifest()` — fetches `reports-manifest.json` and populates the dashboard table
- `applyFilters()` — client-side filter/search logic
- `claudeChat(reportContent, userMessage)` — calls Anthropic API with streaming, appends tokens to chat panel

## File Placement
All generated files go in `html/`. Do not place them anywhere else.
